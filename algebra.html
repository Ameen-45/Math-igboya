Algebra

Algebra is the branch of mathematics that uses symbols and letters to represent numbers and quantities in formulas and equations. It provides the foundation for advanced mathematical concepts and real-world problem solving.

Algebraic Equation Example

2x + 5 = 15

This is a simple linear equation with one variable (x)

You're doing great! Algebra is a powerful tool that will help you solve complex problems.

Key Algebraic Concepts

  • Variables: Symbols (like x, y) that represent unknown values
  • Expressions: Combinations of variables, numbers and operations
  • Equations: Statements that show the equality of two expressions
  • Functions: Relationships between inputs and outputs
  • Polynomials: Expressions with variables raised to whole number exponents

General Linear Equation Form

A linear equation in one variable can be written as:

ax + b = 0

Where:
• a and b are constants
• x is the variable
• a ≠ 0

Types of Algebraic Equations

Linear Equations

First-degree equations that form straight lines when graphed.

y = 2x + 3

3x - 5 = 10

Quadratic Equations

Second-degree equations that form parabolas when graphed.

y = x² - 4x + 4

2x² + 3x - 5 = 0

Polynomial Equations

Equations with terms of different degrees.

y = x³ - 2x² + x - 5

4x⁴ - 16 = 0

Rational Equations

Equations containing fractions with polynomials.

(x + 1)/(x - 2) = 3

1/x + 1/(x + 1) = 5/6

Systems of Equations

Multiple equations with multiple variables.

2x + y = 7

3x - y = 3

Algebraic Operations

Solving Linear Equations

Find the value of the variable that makes the equation true.

Example: Solve 3x - 7 = 14

1. Add 7 to both sides: 3x = 21

2. Divide both sides by 3: x = 7

Factoring Quadratics

Express a quadratic equation as a product of binomials.

Example: Factor x² - 5x + 6

Find two numbers that multiply to 6 and add to -5

Solution: (x - 2)(x - 3) = 0

Quadratic Formula

Solution to any quadratic equation ax² + bx + c = 0:

x = [-b ± √(b² - 4ac)] / 2a

Example: Solve 2x² - 4x - 6 = 0

a = 2, b = -4, c = -6

Solutions: x = 3 and x = -1

Simplifying Expressions

Combine like terms and perform operations.

Example: Simplify 3(x + 2) - 2x + 5

1. Distribute: 3x + 6 - 2x + 5

2. Combine like terms: x + 11

Solving Systems of Equations

Find values that satisfy all equations simultaneously.

Example: Solve the system

2x + y = 7

x - y = -1

Solution: x = 2, y = 3

Algebraic Graphs

Linear Graphs

Equation Form: y = mx + b

m = slope

b = y-intercept

• Straight line graph

• Constant rate of change

Quadratic Graphs

Equation Form: y = ax² + bx + c

a ≠ 0

Vertex form: y = a(x-h)² + k

• Parabolic shape

• Axis of symmetry

Absolute Value

Equation Form: y = a|x - h| + k

Vertex at (h, k)

V-shaped graph

• Always non-negative

• Sharp turning point

Real-World Algebra: Car Loan Calculation

Monthly Payment Formula:

P = [rPV]/[1-(1+r)-n]

Where:

P = Monthly Payment ($471.78)

PV = Loan Amount ($25,000)

r = Monthly Rate (5% APR = 0.05/12)

n = Number of Payments (60 months)

Algebraic Practice Problems

Basic Algebraic Manipulation

  1. Simplify the expression: 4(2x - 3) + 5x
  2. Evaluate 3x² - 2x + 1 when x = -2
  3. Expand and simplify: (x + 3)(x - 5)
  4. Factor completely: x² - 9x + 20
  5. Simplify: (2x² - 3x + 4) - (x² + 2x - 5)

Equation Solving

  1. Solve for x: 5x - 8 = 3x + 4
  2. Find all real solutions: x² + 5x - 14 = 0
  3. Solve: 2(x - 1) = 3(x + 4)
  4. Solve the quadratic equation: x² - 4x - 21 = 0
  5. Solve for x: (x - 2)(x + 6) = 0

Word Problems

  1. The sum of three consecutive integers is 72. Find the integers.
  2. A rectangle has a length that is 3cm more than twice its width. If the perimeter is 54cm, find the dimensions.
  3. A number increased by 6 is equal to 3 times the number. Find the number.
  4. The difference of the squares of two consecutive integers is 15. Find the integers.
  5. If 4 pencils and 3 pens cost $3.50, and 2 pencils and 1 pen cost $1.50, find the cost of one pencil and one pen.

Systems of Equations

  1. Solve the system:

    2x + y = 7

    x - y = -1

  2. Solve the system:

    x + 2y = 8

    3x - y = 5

  3. Solve the system by substitution:

    y = 2x + 1

    3x + y = 10

  4. Solve using elimination:

    4x - 3y = 5

    2x + y = 7

  5. Solve the system:

    x + y + z = 6

    2x - y + z = 3

    x - 2y + 3z = 14
